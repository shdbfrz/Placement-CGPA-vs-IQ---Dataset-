# Placement-CGPA-vs-IQ---Dataset-
A dataset is utilized to develop a classification model predicting student placements based on CGPA. The process includes data preprocessing, exploratory analysis, model training using logistic regression, and deployment on a website. demonstrate ML applications in real-world scenarios, enhancing understanding of model performance and evaluation.
